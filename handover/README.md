A single scrolling page of your work, which you edit yourself through a simple admin screen. No developer needed.

## The short version

- You edit at `ellieparsonagedesign.com/ellieadmin`, signed in with GitHub.
- Changes go live about a minute after you save.
- Running the site is free. The only cost is the domain, renewed once a year.
